# AI Travel Planner 

## About the Project

AI Travel Planner is an AI-powered web application designed to help users plan their trips efficiently through intelligent recommendations and personalized travel assistance. The application leverages Generative AI, RAG (Retrieval-Augmented Generation), and Agentic AI to provide destination suggestions, travel planning guidance, hotel recommendations, and budget-friendly travel options.

## Features

AI-Powered Travel Assistant
Personalized Destination Recommendations
Hotel Suggestions and Travel Guidance
Budget Planning Assistance
PDF Document Processing
Smart Information Retrieval using Vector Database
Interactive User-Friendly Interface

## Technologies Used

Frontend: HTML, CSS
Backend: Flask (Python)
Database: Vector Database
AI Model: Gemini AI
Frameworks & Concepts: RAG (Retrieval-Augmented Generation), Agentic AI

## Project Objectives

1. Develop an intelligent travel planning assistant.
2. Provide personalized travel recommendations based on user preferences.
3. Demonstrate the integration of Generative AI in real-world applications.
4. Improve travel planning through AI-powered decision support.

## How It Works

1. User enters travel preferences or queries.
2. The application processes the request using AI models.
3. Relevant information is retrieved through the vector database.
4. AI generates personalized travel recommendations.
5. Results are displayed to the user in an easy-to-understand format.
